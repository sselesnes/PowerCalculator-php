# ⚡ PowerCalculator — Симулятор енергонезалежності

**PowerCalculator** — це інтерактивний веб-додаток для точного розрахунку часу автономної роботи приміщення при використанні систем резервного живлення (акумулятор + інвертор). Проект враховує реальні фізичні параметри обладнання, такі як глибина розряду (DoD), температурна деградація та пікові пускові навантаження.

---

## 📋 Технічне завдання (ТЗ)

### 1. Концепція продукту

Додаток вирішує проблему складного вибору компонентів резервного живлення для домашніх потреб. Він дозволяє користувачу змоделювати систему, додаючи реальні побутові прилади, та миттєво побачити прогноз автономності.

### 2. Цільова аудиторія

- Домовласники, що планують енергонезалежність.
- Інженери та монтажники систем безперебійного живлення.
- Студенти енергетичних та ІТ-спеціальностей.

### 3. Ключові KPI

- **Точність:** Похибка розрахунку часу роботи < 5%.
- **Зручність:** Можливість отримати готовий звіт у форматі PDF за 1 клік.
- **Надійність:** Збереження даних користувача між сесіями.

---

## 🛠 Функціональні можливості

### 🔋 Розрахунок параметрів АКБ

- **Типізація:** Підтримка LiFePO4 (95% DoD) та AGM (50% DoD).
- **Температурна корекція:** Автоматичне зменшення ємності для AGM акумуляторів при низьких температурах (1% на кожен градус нижче 20°C).
- **Система сповіщень:** Попередження про небезпеку експлуатації LiFePO4 при температурі нижче 0°C.

### ⚡ Моделювання інвертора

- Врахування **ККД (Efficiency)** при розрахунку споживання.
- Моніторинг **номінальної та пікової потужності**.
- Валідація навантаження: візуальне попередження при перевищенні можливостей інвертора.

### 🔌 Конструктор навантаження

- Динамічне додавання/видалення споживачів.
- **База пресетів:** Швидкий вибір типових пристроїв (Холодильник, Роутер, Ноутбук).
- **Пускові струми:** Врахування коефіцієнтів пуску (напр. х5 для холодильників) для аналізу пікового навантаження.

### 📄 Звітність та збереження

- **LocalStorage:** Автоматичне збереження всієї конфігурації в браузері.
- **Smart Print:** Генерація чистого друкованого звіту без елементів інтерфейсу через окреме вікно.

## 📐 Математична модель

Основна формула розрахунку часу автономності ($T$):

$$T = \frac{E_{nom} \cdot V \cdot DoD \cdot K_t \cdot \eta}{P_{total}}$$

Де:

- $E_{nom}$ — номінальна ємність (Ah);
- $V$ — напруга системи (V);
- $DoD$ — коефіцієнт глибини розряду;
- $K_t$ — температурний коефіцієнт;
- $\eta$ — ККД інвертора;
- $P_{total}$ — сумарна потужність активних споживачів (W).
